Attacks

Social Engineering

Social engineering is a type of hack that relies on human interaction and deception to manipulate individuals into divulging confidential information. Examples of social engineering include phishing scams, baiting, and pretexting.

SQL Injection

SQL injection is a type of attack that allows an attacker to inject malicious code into a website's database. This can result in sensitive information such as passwords and credit card numbers being compromised.

Cross-Site Scripting (XSS)

Cross-site scripting (XSS) is a type of security vulnerability that allows an attacker to inject malicious scripts into a web page viewed by other users. This can result in the compromise of sensitive information and the execution of malicious code on the user's device.

Man-in-the-Middle (MitM) Attack

A Man-in-the-Middle (MitM) attack occurs when an attacker intercepts communication between two parties and can eavesdrop on, and even modify, the communication. This can be achieved through various means such as DNS spoofing, ARP spoofing, and SSL hijacking.

Distributed Denial of Service (DDoS) Attack

A Distributed Denial of Service (DDoS) attack is a type of cyber attack that involves flooding a website or network with traffic from multiple sources, making it unavailable to users. This can be achieved through various means such as botnets, amplification attacks, and reflection attacks.
